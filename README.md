# Algorithm-for-reconstructing-a-tree-based-on-a-set-of-distances
This project is the first part of a physics project to reconstruct tree-type graphs.

Если вы всё таки решитесь скачать Jupyter файл или файл Python, то с ними следует работать следующим образом:
  1. Устанавливаете Jupyter, scipy, numpy, matplotlib, networkx
  2. Создаёте файл matrix.txt и записываете в него матрицу расстояний в предложенном формате (смотри файл-пример)
  3. Запускаете код в Jupyter файле или просто запускаете Python файл (эти два файла эквивалентны в плане функционала)

Вы всегда можете скачать release, который находится справа в web версии на ПК. В таком случае ничего устанавливать дополнительно не нужно.
